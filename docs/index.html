aprendeFaker

Datos sintéticos reproducibles para enseñar ensamblador ARM64 y el stack IoT — sin hardware, autoevaluables byte a byte.

EN — Reproducible synthetic-data generators for teaching ARM64 assembly and the IoT stack (MQTT, InfluxDB, Grafana, MongoDB, SQL Server, ML). One CLI, self-grading labs, no hardware required. Docs in Spanish.

Inicio rápido (AWS Academy — Ubuntu Server)

git clone https://github.com/tectijuana/aprendeFaker.git && cd aprendeFaker
python3 -m venv .venv && .venv/bin/pip install -r requirements.txt
.venv/bin/python aprendefaker.py            # lista los subcomandos
.venv/bin/python aprendefaker.py influx --seed 2026

Regla de oro: todo se genera con semilla. Misma semilla → mismos bytes → tu salida se compara contra la esperada con cmp. Si no coincide, el error está en tu programa, no en los datos.
