# Código de conducta

Este proyecto adopta el [Contributor Covenant v2.1](https://www.contributor-covenant.org/es/version/2/1/code_of_conduct/)
(versión en español).

En corto: este es un espacio académico. Se participa con respeto, se critica el
código y no a las personas, y se recibe a quien empieza con la misma seriedad que
a quien ya sabe. Sin acoso, sin discriminación, sin excepciones.

Los incidentes se reportan de forma confidencial a la persona responsable del
curso: **rene.solis@tectijuana.edu.mx**. Todo reporte se revisa y se responde.
